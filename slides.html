Welcome!

What is version control?

  • Any system that manages changes to files over time.
  • Benefits of a version control system include:
    • Collaboration
    • Versioning

Before we begin…

$ pwd

/Users/username/Documents/my-directory



$ ls

README.md

Before we begin…

$ cat README.md

### Hi, my name is…

* bullet point 1
* bullet point 2

Before we begin…

$ git config --list

user.email=my.name@mcgill.ca
...
...

If necessary…

$ git config --global user.name "My Name"

$ git config --global user.email "my.name@mcgill.ca"

Now we’re ready!

$ git init -b main

Now we’re ready!

$ git remote add origin https://github.com/MY_USERNAME/MY_USERNAME.git

Now we’re ready!

$ git add README.md

Now we’re ready!

$ git commit -m "First commit"

Now we’re ready!

$ git push -u origin main
